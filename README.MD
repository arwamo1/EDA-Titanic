# Titanic EDA Project

## Description
This project applies Exploratory Data Analysis (EDA) on the Titanic dataset.

## Data Cleaning
- Dropped Cabin column
- Filled missing Age values using median
- Filled missing Embarked values using mode

## Visualizations
- Survival Count
- Survival by Gender
- Survival by Class
- Age Distribution
- Age vs Survival
- Survival by Embarked Port
- Average Fare by Class
